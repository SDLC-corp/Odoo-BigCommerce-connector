# BigCommerce Connector

Production-oriented base scaffold for an Odoo 18 addon that will host BigCommerce integration components.

## Current Scope

- Module metadata and dependency baseline
- Security groups for user and manager roles
- Top-level BigCommerce menu and placeholder child menus
- Empty package initializers for modular expansion (`models`, `services`, `controllers`, `wizard`)

## Not Included Yet

- API client implementation
- Synchronization services
- Webhook endpoints
- Business models and workflows
